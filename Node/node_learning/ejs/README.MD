## ejs

特性

`<% %>` 用于控制流

`<%= %>` 用于转义的输出

`<%- %>`用于非转义的输出

`-%>` 结束标签用于换行移除模式

带有`<%_ _%>`的控制流使用空白字符移除模式

自定义分隔符 (例如，使用 `'<? ?>'` 代替 `'<% %>'`)
